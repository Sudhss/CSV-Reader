# CSV Reader 

Ever get a janky CSV file and just wanna see the data without all the headache? This bad boy takes your uploaded CSV and gives you a clean, easy-to-read table right away. No cap.

###  Features

* **Effortless Upload:** Just drag and drop your `.csv` file.
* **Clean Output:** Turns your raw data into a slick, organized table.
* **Simple AF:** No complex steps, just pure vibes.

###  How to Use

1.  Hit the "Upload" button and pick your `.csv` file.
2.  The data will automatically show up in a neat table. It's that simple.
